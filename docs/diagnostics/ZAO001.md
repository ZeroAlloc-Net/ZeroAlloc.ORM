# ZAO001 — Annotated method must be partial

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Trigger

Fires when a method carries a generator attribute such as `[Query]` but its declaration is missing the `partial` modifier. The source generator emits the method body in a sibling partial file, so the user-authored side must be a partial declaration with no body.

## Fix

Add the `partial` modifier to the method declaration and remove any inline body — the generator supplies the implementation. The containing type must also be `partial` (see ZAO004).

## Example

```csharp
// Fires ZAO001:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public Task<int> GetOneAsync(CancellationToken ct);  // missing 'partial'
}

// Fix:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public partial Task<int> GetOneAsync(CancellationToken ct);
}
```
