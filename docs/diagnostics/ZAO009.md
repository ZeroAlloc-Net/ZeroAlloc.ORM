# ZAO009 — Redundant async keyword on generated partial

**Severity:** Warning
**Category:** ZeroAlloc.ORM

## Trigger

Fires when a generator-annotated partial method is declared with the `async` modifier. The generator emits the async state machine in the implementing partial, so the user-authored declaring partial must not be `async` — the compiler would refuse to match the two halves.

## Fix

Remove the `async` keyword from the partial declaration. The method still returns `Task<T>`/`ValueTask<T>` and is awaitable; the generator handles the state machine.

## Example

```csharp
// Fires ZAO009:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public async partial Task<int> GetOneAsync(CancellationToken ct);  // 'async' redundant
}

// Fix:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public partial Task<int> GetOneAsync(CancellationToken ct);
}
```
