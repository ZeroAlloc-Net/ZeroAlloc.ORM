# ZAO042 — [StoreAsString] requires an enum type

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Trigger

Fires when `[StoreAsString]` is applied to a type that is not an enum. The attribute exists to switch enum round-trip from the underlying integral type to the member name (string). It has no meaning on classes, structs, records, or interfaces.

## Fix

Either remove `[StoreAsString]`, or move it onto an `enum` declaration.

## Example

```csharp
// Fires ZAO042 — [StoreAsString] on non-enum type:
[StoreAsString]
public sealed class NotAnEnum {}
```

Fix by applying it to an enum:

```csharp
[StoreAsString]
public enum OrderStatus
{
    Pending,
    Shipped,
    Cancelled,
}
```
