# ZAO044 — Ambiguous convention discovery

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Status

**Reserved** — forward-looking safety net. The strict priority order in `ConventionDiscovery.Resolve` (Materialize > Primitive > ValueObject > StaticFactory > SingleArgCtor > Enum > MultiArgCtor > Unknown) prevents most ambiguity from arising in practice, so the descriptor exists today against future edge cases (custom convention plugins, partial-source generators, etc.) without an active emit path.

## Trigger (future)

Will fire when a type matches multiple convention rules with equal priority and no clear precedence — for example, a type carrying both `[ValueObject]` and a hand-rolled `static From` whose semantics conflict.

## Fix (future)

Add an explicit `[Materialize(Strategy = ...)]` to pick one branch unambiguously.
