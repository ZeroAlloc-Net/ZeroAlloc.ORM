# ZAO002 — Unsupported return type

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Trigger

Fires when a `[Query]`-annotated method has a return type the generator cannot emit. Allowed shapes are `Task<T>`, `ValueTask<T>`, `IAsyncEnumerable<T>`, `Task`, and `ValueTask`. Synchronous return types (`T`, `IEnumerable<T>`) and awaitable types other than the listed ones are rejected.

## Fix

Change the return type to one of the supported async shapes. For a single scalar or row, use `Task<T>` or `ValueTask<T>`. For streaming results, use `IAsyncEnumerable<T>` (see ZAO007 for the `[EnumeratorCancellation]` requirement). For commands that return no rows, use `Task` or `ValueTask`.

## Example

```csharp
// Fires ZAO002:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public partial int GetOneAsync(CancellationToken ct);  // sync return type
}

// Fix:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public partial Task<int> GetOneAsync(CancellationToken ct);
}
```
