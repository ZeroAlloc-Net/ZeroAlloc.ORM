# ZAO007 — IAsyncEnumerable<T> return without [EnumeratorCancellation]

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Trigger

Fires when a `[Query]`-annotated method returns `IAsyncEnumerable<T>` but no `CancellationToken` parameter is marked `[EnumeratorCancellation]`. Without that attribute the C# compiler does not forward the consumer-supplied `WithCancellation` token to the iterator, so cancellation cannot propagate into the data reader loop.

> Note: v0.1 does not yet emit `IAsyncEnumerable<T>` bodies (streaming is deferred). This diagnostic ships as a forward-looking guard.

## Fix

Add `[EnumeratorCancellation]` (from `System.Runtime.CompilerServices`) to the `CancellationToken` parameter.

## Example

```csharp
// Fires ZAO007:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT id FROM t")]
    public partial IAsyncEnumerable<int> StreamAsync(CancellationToken ct);  // missing attr
}

// Fix:
using System.Runtime.CompilerServices;

public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT id FROM t")]
    public partial IAsyncEnumerable<int> StreamAsync([EnumeratorCancellation] CancellationToken ct);
}
```
