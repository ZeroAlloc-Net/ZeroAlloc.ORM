# ZAO043 — [Materialize(Factory)] references missing method

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Status

**Active** — fires in v0.5 Phase D whenever a `[Materialize(Factory = "X")]` annotation cannot be resolved into a callable static factory method.

## Trigger

Fires when a method (or its return-type / parameter-type) carries `[Materialize(Factory = "Name")]` and one of the following holds:

- The named method does not exist on the target type.
- The named method exists but is not `static`.
- The named method exists but is not `public` (or `internal`).
- A factory parameter type cannot be resolved by ConventionDiscovery.
- `[Materialize(Strategy = Custom)]` is present but `Factory` is unset.

The diagnostic message carries the failure reason verbatim so adopters see the specific cause without ambiguity.

## Fix

Pick the fix that matches the reported reason:

- "method not found" — correct the spelling, or add the static factory method.
- "method is not static" — add the `static` modifier.
- "method is not public" — change accessibility to `public` (or `internal`).
- "factory parameter type '<TypeName>' could not be resolved by ConventionDiscovery" — make the parameter type a primitive / value-object / single-arg ctor / static-factory convention match, or add an explicit `[Materialize(Factory)]` on the inner type.
- "[Materialize(Strategy = Custom)] requires a Factory argument" — supply the `Factory = "..."` named argument, or remove the explicit `Strategy = Custom`.

## Example

```csharp
// Fires ZAO043 — reason "method not found":
[Materialize(Factory = "Create")]
public sealed record Money(decimal Amount, string Currency);
// (Money has no static Create method)

// Fires ZAO043 — reason "method is not static":
[Materialize(Factory = "From")]
public sealed record Money(decimal Amount, string Currency)
{
    public Money From(decimal a, string c) => new(a, c); // missing 'static'
}

// Fix — supply the static factory:
[Materialize(Factory = "From")]
public sealed record Money(decimal Amount, string Currency)
{
    public static Money From(decimal amount, string currency) => new(amount, currency);
}
```
