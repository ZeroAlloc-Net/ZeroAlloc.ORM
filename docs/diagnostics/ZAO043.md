# ZAO043 — [Materialize(Factory)] references missing method

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Status

**Reserved** — wired in v0.5 when `[Materialize]` is re-introduced. The descriptor exists today so the `helpLinkUri` resolves pre-v0.5 and the analyzer release tracking lists the code in its canonical order.

## Trigger (v0.5)

Will fire when a method carries `[Materialize(Factory = "Name")]` but the named factory either:

- Does not exist on the target type.
- Is not `static`.
- Is not `public`.
- Has a parameter list incompatible with the column shape.

## Fix (v0.5)

Either correct the factory name, mark the factory `public static`, or remove the `Factory` argument and let convention discovery pick a strategy.
