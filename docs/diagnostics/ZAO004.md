# ZAO004 — Containing type must be partial

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Trigger

Fires when a type contains generator-annotated methods but the type itself is not declared `partial`. The generator emits a sibling partial declaration alongside the user-authored file, so both halves must be `partial`.

## Fix

Add the `partial` modifier to the containing class, struct, or record declaration.

## Example

```csharp
// Fires ZAO004:
public sealed class Repo(IAsyncDbConnection connection)  // missing 'partial'
{
    [Query("SELECT 1")]
    public partial Task<int> GetOneAsync(CancellationToken ct);
}

// Fix:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public partial Task<int> GetOneAsync(CancellationToken ct);
}
```
