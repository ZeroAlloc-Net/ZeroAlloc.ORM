# ZAO003 — No IAsyncDbConnection found on containing type

**Severity:** Error
**Category:** ZeroAlloc.ORM

## Trigger

Fires when a type contains `[Query]`-annotated methods but the generator cannot find an `IAsyncDbConnection` reference to wire the command against. The generator looks for a primary-constructor parameter, an instance field, or an instance property of type `IAsyncDbConnection`.

## Fix

Inject `IAsyncDbConnection` into the type via a primary constructor parameter (recommended), or add a field/property of that type. The generator picks the first match it encounters.

## Example

```csharp
// Fires ZAO003:
public sealed partial class Repo
{
    [Query("SELECT 1")]
    public partial Task<int> GetOneAsync(CancellationToken ct);  // no connection
}

// Fix:
public sealed partial class Repo(IAsyncDbConnection connection)
{
    [Query("SELECT 1")]
    public partial Task<int> GetOneAsync(CancellationToken ct);
}
```
