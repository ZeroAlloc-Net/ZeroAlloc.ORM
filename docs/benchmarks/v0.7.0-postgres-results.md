# v0.7.0 Postgres benchmark results

**Status:** _Pending local capture._ Postgres benchmarks require Docker
(Testcontainers boots `postgres:16-alpine`) and are not run in CI to keep
PR latency low. Capture on your own machine and paste the four
`*-report-github.md` tables under the headings below.

## Reproduction

```bash
# Docker must be reachable.
dotnet run -c Release --project tests/ZeroAlloc.ORM.Benchmarks -- \
  --filter "*Postgres*" --exporters markdown json
```

Or by category:

```bash
dotnet run -c Release --project tests/ZeroAlloc.ORM.Benchmarks -- \
  --anyCategories Postgres
```

Each fixture boots a fresh container, so the run is markedly slower than
Sqlite — budget ~10× wall-clock for the full Postgres suite.
