captures are reference points, not specifications. ZA.ORM is a
**relative-position** tool: trust ratios across rows in the same table
rather than absolute throughput.

## Reproduction

```bash
# From repo root
dotnet build -c Release tests/ZeroAlloc.ORM.Benchmarks
dotnet run -c Release --project tests/ZeroAlloc.ORM.Benchmarks -- \
  --filter "ZeroAlloc.ORM.Benchmarks.SingleRowReadBench*" \
           "ZeroAlloc.ORM.Benchmarks.MultiRowReadBench*" \
           "ZeroAlloc.ORM.Benchmarks.MultiResultSetBench*" \
           "ZeroAlloc.ORM.Benchmarks.InsertBench*" \
  --exporters markdown json
```

The exporter drops a markdown summary under
`tests/ZeroAlloc.ORM.Benchmarks/BenchmarkDotNet.Artifacts/results/` — copy
the four `*-report-github.md` tables here under the headings below.
